# Pali Script Converter

A web-based Pali script transliteration/conversion tool built using Flask, Python, HTML, CSS, and JavaScript.

Supports conversion between multiple Indic and Asian scripts.

---

# Supported Scripts

- Sinhala
- Devanagari
- Roman
- Thai
- Lao
- Myanmar
- Khmer
- Bengali
- Gurmukhi
- Telugu
- Kannada
- Malayalam
- Tibetan
- Cyrillic

---

# Features

- Real-time script conversion
- Clean web interface
- Multiple script support
- Flask backend
- Easy to extend

# Installation

Clone repository:

```bash
git clone https://github.com/Pankaj27-prog/ScriptConverter.git
```

Move into project folder:

```bash
cd ScriptConverter
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# Run Project

Start Flask server:

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
